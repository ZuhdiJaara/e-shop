# E-Shop

## Mini e-commerce project using a fake API

I implemented the DummyJSON API using the "Alamofire" and "SwiftyJSON" libraries. I needed to use a few libraries, such as "Cosmos" to convert the rating from Double to Stars, the "SideMenu" library to create a side menu on the home page, and finally the "NVActivityIndicatorView" library to display the loader view when the data is loaded. All of these libraries were implemented using Cocoapods.

First, I created the login page, then I created three main pages and merged them into the UITabBarController: **Home page**, **Categories page**, and **Cart page**. The **Products page** contains products based on the page from which it comes, so it can contain all products or products based on category. There is also a page containing **Product details** such as dimensions and shipping information, and the **Reviews page** that contains product ratings with comments.
